# libcollections
A C library with some generic collections, strings and a few more...

## License

LGPLv2

## Contents

* INI configuration
* JSON parser
* Datetime functions
* String manipulation (also string lists)
* Doubly linked list
* Thread utils
* Timer functions
* IPC socket protocol
* And some more..

## Dependencies

* Pthreads
* Realtime extensions library
* python-enum34 for python extension
* libmagic
* Java 1.6 JDK
